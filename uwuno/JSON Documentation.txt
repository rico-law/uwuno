### ROOMS ###
POST
http://localhost:8080/rooms
{
    roomName
}

GET
http://localhost:8080/rooms

PUT
http://localhost:8080/rooms/<uid>
{
    uid,
    roomName
}

DELETE
http://localhost:8080/rooms/<uid>
